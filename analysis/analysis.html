# This function will compute percentages and confidence intervals for specific variables and format them.

count_prop_ci <- function(data, ..., ci_method) {
  columns <- enquos(...)

  data <- data %>%
      count(!!!columns, .drop = FALSE) 
  
  data <- data %>%
      cbind(MultinomCI(data$n, conf.level = 0.95, sides = "two.sided", method=ci_method)) %>%
      mutate(N = sum(n)) %>%
    select(!!!columns, n, N, percent = est, ci_lwr = lwr.ci, ci_upr = upr.ci) %>%
    mutate(percent = percent*100, ci_lwr = ci_lwr*100, ci_upr = ci_upr*100, 
           report = paste0(n,' (',signif(percent,2),'% [',signif(ci_lwr,2),'%-',signif(ci_upr,2),'%])'))
  
  return(data)
}

# Same function as above but broken down by group (e.g., year)
count_prop_ci_by <- function(data, ..., group, ci_method) {
  cols <- enquos(...)
  grp  <- enquo(group)
  data %>%
    group_by(!!grp) %>%
    count(!!!cols, .drop = FALSE) %>%
    group_modify(~ {
      ci <- DescTools::MultinomCI(.x$n, conf.level = 0.95, sides = "two.sided", method = ci_method) %>%
        as_tibble()
      bind_cols(.x, ci) %>%
        mutate(
          N = sum(n),
          percent = est * 100,
          ci_lwr = lwr.ci * 100,
          ci_upr = upr.ci * 100,
          report = paste0(n, " (", signif(percent, 2), "% [", signif(ci_lwr, 2), "%–", signif(ci_upr, 2), "%])")
        )
    }) %>%
    ungroup() %>%
    select(!!grp, !!!cols, n, N, percent, ci_lwr, ci_upr, report)
}

The vast majority of articles did not have a preregistration; however, the prevalence of preregistration has increased modestly over time.

Preregistration statements

525 (89% [87%-92%]) articles did not contain a statement about preregistration, 16 (2.7% [0.51%-5.2%]) articles stated that there was no preregistration, and the remaining 48 (8.1% [5.9%-11%]) articles stated that there was a preregistration.

Functional preregistration

Of the 48 articles that stated there was a preregistration, we could not access the preregistration for 5 articles, due to broken links (n = 4) or unclear access instructions (n = 1). Thus, the overall prevalence of functional preregistration was 43 (7.3% [5.5%-9.7%]) articles.

Use of preregistration over time

There was a modest increase in functional preregistration over time (Figure 1). Specifically, we identified functional preregistrations for 9 (4.6% [2.4%–8.5%]) articles in 2012, 11 (5.7% [3.2%–9.9%]) articles in 2018, and 23 (11% [7.7%–17%]) articles in 2024. A post-hoc (not preregistered) chi-square test indicated a statistically significant increase in functional preregistration between 2012 and 2024, χ²(1, N = 396) = 6.21, p = .01.

Preregistration registries

The most commonly used registry for functional preregistration was ClinicalTrials.gov (n = 29 articles) with other platforms each accounting for less than 5 articles (for details, see Supplementary Table Z). We did not find any Registered Reports.

A post-hoc (not preregistered) chi-square test indicated a statistically significant increase in the use of reporting guidelines between 2012 and 2024, χ²(1, N = 396) = 15.44, p < .001.

  1. Measures — if there is disagreement about measures availability, then I check. In my coding, there has to be an explicit statement of availability of the original materials, it’s not sufficient to just mention the source of the measure. A brief or incomplete description of a measurement instrument did not suffice.
