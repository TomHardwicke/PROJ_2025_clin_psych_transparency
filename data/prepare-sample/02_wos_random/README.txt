# Information about how these data files were obtained

## Files: wos_2012_random.csv, wos_2018_random.csv, wos_2024_random.csv

Person who obtained the data: Tom Hardwicke 

NOTE: this data file cannot be publicly shared because it contains large portions of a proprietary database.

Instructions: 

1. These data files were created using the prepare-articles.r script.
2. The script removes articles WoS has identified as being retracted (2012, n=2; 2018, n = 3; 2024, n = 2) then randomly samples 400 articles (in each data file).
