coding_differences.xlsx was created by running the script coder_differences_main.R on data_bianca.csv and data_justine.csv in the "primary" data folder. In coding_differences.xlsx each article has a row of data extraction by Bianca and one by Justine, with a third "consensus" row. The consensus row highlights the differences between Bianca and Justine's coding and those differences have not yet been resolved.

coding_differences_resolved.xlsx was manually created by duplicating coding_differences.xlsx on 2026-01-09. TEH worked through the coding discrepancies (highlighted in yellow) and manually resolved them in the rows with "Consensus" in the coder column so that eventually the consensus row represented the finalised coding to be used in the main data analysis.

TEH also added a column "data_access_restrictions_type" which he used to classify data access restrictions into particular types.
 
After these manual changes, coding_differences_resolved.xlsx was locked on 2026-03-09 and no further changes were made.
